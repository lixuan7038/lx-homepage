# lx-homepage
开源代码 黎轩个人主页源码 纯静态页面
### 宝塔服务器部署
1. 上传文件 ：将所有文件上传到网站目录
2. 设置网站 ：
   - 域名：你的域名（如 lixuan.cyou ）
   - 根目录：网站目录路径
   - 无需PHP/数据库
3. 访问 ：打开浏览器访问你的域名

项目格式
myhome/
├── index.html    # 主页面
├── css/style.css # 样式文件
├── js/script.js  # 交互脚本
└── image/        # 图片资源
